## 📖 Quran Web App – Frontend

A simple, responsive Quran reading web application built with Next.js.
Users can browse all surahs, read ayahs with translation, and customize reading settings for a better experience.

### 🌐 Live Link: https://quran-application-blue.vercel.app/

---
#### 🛠 Technology : **Next.js**, **TypeScript**,  **Shadcn UI**, **Context API**.

#### ✨ Features :

* Responsive UI with sidebar (desktop) & drawer (mobile)
* Surah list page (Arabic + English names)
* Surah details page with full ayahs (Arabic + translation)
* Search ayahs by translation text
* Settings panel:

  * Change Arabic font (multiple options)
  * Adjust Arabic font size
  * Adjust translation font size
  * Persistent settings (localStorage)
* Smooth loading experience using Skeleton

---

#### ⚙️ How to Set Up Locally

* Make sure **Node.js** and **Git** are installed on your machine

* Clone the repository
  `git clone https://github.com/Rockrayhan/Quran-Web-Application`

* Navigate to the project folder and run:
  `npm install`
  `npm run dev`

* Open your browser and visit:
  `http://localhost:3000/`
